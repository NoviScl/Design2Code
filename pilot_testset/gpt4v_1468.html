What is char octal escape sequence?

A character literal can be expressed as an escape sequence in the form '\\nnn'. 'n' is an octal digit (0-7). The range for the octal escape sequence is '\\000' to '\\377'. The octal number 377 is the same as the decimal number 255.

Using octal escape sequence, you can represent characters whose Unicode code range from 0 to 255 decimal integers.

char c2 = '\\142';
char c3 = '\\102'; // Same as 'b'
//char c4 = '\\400'; // A compile-time error. Octal 400 is out of range

Demo

public class Main {
    public static void main(String[] args) {
        char c1 = '\\u0021';
        char c2 = '\\142';
        char c3 = '\\102'; // A compile-time error. Octal 400 is out of range
        char c4 = 'b';     // Same as '\\142'

        System.out.println(c1); //from '!' to 'y' is 32 to 121 in octal
        System.out.println(c2);
        System.out.println(c3);
        System.out.println(c4);
    }
}
