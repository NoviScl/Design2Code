Installation & Configuration

CallScripter is a web application that is hosted on a Microsoft Web Server (IIS) and uses Microsoft SQL server database to store all the information and data collected by the application. Some features within the application require supporting windows services. The CallScripter software components are deployed across multiple servers that collectively make up the system. To implement CallScripter we recommend at least two dedicated servers:
 
    Microsoft IIS web server
    Microsoft SQL server to host the CallScripter database
 
Note: The database and website can reside on the same server but in many cases dedicated hardware is recommended to provide the best level of application performance.
 
CallScripter 4.5.45 CIC integration references have been updated to 2015 R3.
