50 watt Large Marking Breadth Fiber Laser Marking Equipment For 3c Industry
 
1. Machine characteristics:
 
This machine with X, Y axis moving working table which can achieve the big breadth laser marking,  using the internal air-cooling system, fiber output, flexible application and easy to operate.
 
Can mark many pieces of work piece at the same time automatic which high improve the production efficiency.
 
This automatic laser marking machine's working procedures as follows:
 
Loading the work piece on the big working table⇒positioning each workpiece⇒start the laser marking⇒the laser will do the laser marking on each work piece which already loaded on the table automatic⇒unload the marked work piece.
 
PS: how many work piece can be loaded on the working table depend on the work piece size
