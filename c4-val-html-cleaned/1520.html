Date : 1968

Location : NAP/LIRN

Photo : collection Skystef

Description : Sabena ninth Caravelle was delivered on December 29 1964. It remained operational for the carrier till June 1972 and is here captured in the first (and airframes only) livery.

Date : 1972

Location : BRU/EBBR

Photo : collection Skystef

Date : 10/1973

Location : LIN/LIML

Photo : collection Skystef

Description : second livery adopted in January 1973.

Date : 10/1978

Location : -

Photo : collection Skystef

Description : basic second color scheme without titles after a lease to Catair between April 1976 and January 1978. It briefly resumed operations for Sobelair in the summer of 1978.
