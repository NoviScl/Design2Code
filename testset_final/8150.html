Who Was Helen Keller?

Written by Gare Thompson

Illustrated by Nancy Harrison


Reviewed by Sabella M. (age 9)


If you want to be really sad and emotional, read Who Was Helen Keller. Helen was deaf and blind.

In my opinion, this book will make you feel many emotions. I believe this is a sad book because Helen is sick.

I recommended this book, if you enjoy helping people. Also, if you want learn about sign language then read this book.

Sabella M.
is a student in
Ms. Tierney's 3rd Grade Class
